
Axera has been deployed to namespace "{{ .Values.global.namespace }}" (release "{{ .Release.Name }}").

1. Watch the workloads come up:
     oc get pods -n {{ .Values.global.namespace }} -w

   First boot runs database migration Jobs{{ if not .Values.ai.external }} and (if AI is enabled) an Ollama model-pull Job{{ end }}; give it a few minutes.

2. Open the console (frontend Route):
{{- $fhost := include "axera.frontendHost" . }}
{{- if $fhost }}
     https://{{ $fhost }}
{{- else }}
     oc get route frontend -n {{ .Values.global.namespace }} -o jsonpath='{.spec.host}{"\n"}'
   (OpenShift auto-assigned the hostname from its default domain.)
{{- end }}

3. Sign in with the bootstrap admin account:
     Username: {{ .Values.app.adminUsername }}
{{- if .Values.app.adminPassword }}
     Password: (the value you set in app.adminPassword)
{{- else }}
     Password: (auto-generated) retrieve it with:
       oc get secret axera-app-secret -n {{ .Values.global.namespace }} \
         -o jsonpath='{.data.AdminCredentials__Password}' | base64 -d ; echo
{{- end }}
   Change this password after first login.

4. Licensing: a {{ .Values.licensing.trial.days }}-day trial is seeded automatically. Upload a
   production license under Admin → License before it expires.
